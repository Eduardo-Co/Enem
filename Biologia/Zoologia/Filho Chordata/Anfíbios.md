## Sistemas

| Eixo           | Essencial                                                                |
| -------------- | ------------------------------------------------------------------------ |
| **Esqueleto**  | Endoesqueleto ósseo                                                      |
| **Respiração** | Branquial (larvas) → pulmonar + cutânea (adultos)                        |
